------------------------------------------------------------
1. Installed AWS CLI v2
------------------------------------------------------------
# Download and unzip
curl "https://awscli.amazonaws.com/awscli-exe-linux-x86_64.zip" -o "awscliv2.zip"
unzip awscliv2.zip

# Install
sudo ./aws/install

# Confirm installation
aws --version

✅ Outcome: Installed AWS CLI v2 successfully.

------------------------------------------------------------
2. Installed AWS IAM Authenticator
------------------------------------------------------------
# Download binary

# Make it executable and move it to PATH
chmod +x aws-iam-authenticator
sudo mv aws-iam-authenticator /usr/local/bin/

# Confirm version
aws-iam-authenticator version

✅ Outcome: Authenticator binary installed in /usr/local/bin.
